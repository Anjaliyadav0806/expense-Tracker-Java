Expense Tracker (Java)

📌Project Description

This is a console-based Expense Tracker application developed using Java. It allows users to manage their daily expenses by adding, viewing, and deleting expense records in a simple menu-driven interface.

Features

- Add expense with category and amount
- View all expenses with date tracking
- Delete expense by category
- Category-wise expense summary
- Input validation for amount
- Real-time date tracking using LocalDate

Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList (Data storage)
- HashMap (Category-wise summary)
- LocalDate (Date handling)

How to Run

1. Clone or download the project
2. Open in any Java IDE (VS Code / IntelliJ / Eclipse)
3. Compile and run "ExpenseTracker.java"
4. Use the menu options to interact
